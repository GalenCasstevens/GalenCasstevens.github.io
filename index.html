Battleship Game

I created the Battleship game with JavaScript and a little HTML and CSS. You type in a location in the form in the bottom right and hit enter or press fire, then a message appears in the top left indicating whether your guess was a hit, miss, or sank a ship. When the game is over you are told the number of guesses it took you took you to win. The code implements a model-view-controller design pattern.
